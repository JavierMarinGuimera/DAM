Security-related files

This directory contains some documents and programs related to computer security. They have been collected from several sites, and I do not take any responsibility for them. However, I try to maintain this archive and update its contents to newer versions, if I am aware of their existence. Below you will find descriptions of some important files:
  • ANNOUNCE - a description of TAMU security package
  • netlog.README - description of NETLOG package (part of TAMU)
  • INDEX - an index to files from the University of Utrecht
  • cert/ - CERT advisories on known security holes
  • firewalls/ - Brent Chapman's selections from firewalls mailing-list
  • libdes.93-10-08.tar.gz - very fast DES and crypt() by Eric Young
  • pgp2.3A/ - PGP 2.3A public key encryption package
  • phrack/ - old issues of Phrack magazine
  • netlog.README - description of NETLOG package
  • tcp_wrapper.txt.gz - description of Wietse Venema's TCP wrapper
  • tiger.README - description of Tiger security-checking package
  • hack.faq - IRC #hack FAQ
  • Security-SunOS4.txt - how to make a SunOS 4.1.3 machine more secure, this document is mirrored from here.
Please note the several files with names containing "npasswd" - those are different versions of /bin/passwd replacement:
  • npasswd.tar.gz - the original version by Clyde Hoover
  • npasswd_jpl.tar.gz - improved at JPL
  • szs_npasswd.tar.gz - hacked by me to support NIS & C2 on SunOS
  • szs_npasswd-1.tar.gz - further hacked by tsurmacz@ict.pwr.wroc.pl
